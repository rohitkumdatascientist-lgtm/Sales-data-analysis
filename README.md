📊 Linear Regression Project – Sales Prediction
📌 Project Overview

This project demonstrates the implementation of a Linear Regression model using Python to predict sales (Item_Outlet_Sales) based on training data. It covers the full machine learning workflow including data loading, preprocessing, model training, and evaluation.

🚀 Features
Load and explore training & testing datasets
Separate features (X) and target variable (y)
Train a Linear Regression model using scikit-learn
Generate predictions on both training and test data
Evaluate model performance using RMSE (Root Mean Squared Error)
Understand model coefficients and intercept
